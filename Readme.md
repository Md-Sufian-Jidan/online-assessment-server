**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT
- CORS
- Cookies

```bash
git clone https://github.com/Md-Sufian-Jidan/online-assessment-server.git
cd online-assessment-server

npm install

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

npm run start

- The API will be running at:

http://localhost:5000

# Build
npm run build
